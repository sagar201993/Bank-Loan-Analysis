# Bank Loan Analysis – Complete End-to-End Power BI & SQL Project 

This project delivers a complete Data Analyst solution using:

- **SQL Server**
- **Power BI**
- **Power Query**
- **DAX**
- **Data Modelling**
- **Dashboard Design**
- **Finance Domain Knowledge**

The goal is to analyze the performance of bank loans, borrower behavior, risk exposure, repayment patterns, and financial KPIs that help lenders make business decisions.

---


---

##  Project Overview

Banks process large volumes of loan applications daily.  
This project helps the bank:

- Track loan applications & disbursements  
- Monitor repayments & cash flow  
- Identify risky loans  
- Analyze borrower behavior  
- Measure portfolio performance  
- Compare good vs bad loans  
- Discover trends by time, region, purpose, and employment

#  Dataset Terminologies

Definitions from **Terminologies in Data.docx**:

- **Loan Amount** – Total principal disbursed  
- **Interest Rate** – Cost of borrowing  
- **DTI** – Borrower debt stress  
- **Funded Amount** – Actual disbursal  
- **Total Received** – Amount repaid  
- **Loan Status** – Performance indicator  
- **Purpose** – Borrower’s loan reason  
- **Home Ownership** – Collateral indicator  
- **Employment Length** – Stability indicator  
- **Grade/Subgrade** – Credit risk category

#  End-to-End Workflow

1. Load raw CSV into SQL  
2. Clean & validate data  
3. Build SQL queries for insights  
4. Load into Power BI  
5. Create date table  
6. Build model relationships  
7. Write DAX measures  
8. Build 3 dashboards  
9. Format UI & navigation  
10. Upload complete project to GitHub
